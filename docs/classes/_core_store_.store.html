Class Store<S, A>

Provides state storage that mutates when some action happened. State mutates by reducers - functions that mutates state in respond to action. This entity usually used to centralized manipulate the state and notify different parts of the application about state change.

Type parameters

  • S

    Specifies the shape of state object.

  • A: IAction

    Specifies the shape of action object.

  • addReducers(...reducers: Array<StoreReducer>): void
    • Defined in Core/Store.ts:58

    Attaches state reducers to current store instance.

    example

    Add reducer (function that mutates the store) to the store instance.

    store.addReducers((state: StoreState, action: StoreAction) => {
    switch (action.type) {
        case: StoreActionType.Reset:
            return {
                items: [],
                page: 0
            };
    }

    return state;
});
