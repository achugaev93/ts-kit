# Core

Provides core functionality: collections, exceptions, comparison, random and text utils

Related links:

- [API reference](https://monumentjs.github.io/package/core)

# Install

Install with NPM:

```
npm i @monument/core
```

Install with Yarn:

```
yarn add @monument/core
```
